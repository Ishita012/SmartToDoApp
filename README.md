# Smart Todo App

A simple full-stack Todo application with user authentication using JWT.

## Features
- User Signup & Login
- JWT Authentication
- Add and Delete Tasks
- Backend using Node.js & Express

## Tech Stack
- Frontend: HTML, JavaScript  
- Backend: Node.js, Express  
- Auth: JSON Web Token (JWT)

## How to Run

1. Install dependencies
2. Start server
3. Open `index.html` in browser

## API Routes
- POST /api/register  
- POST /api/login  
- POST /api/add-task  
- GET /api/tasks  
- POST /api/delete-task  

## Note
- Data is stored temporarily (no database)
